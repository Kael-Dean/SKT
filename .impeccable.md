## Design Context

### Users
พนักงานภายในสหกรณ์การเกษตรเพื่อการตลาดลูกค้า ธ.ก.ส. สุรินทร์ (AMC Surin) — 5 กลุ่ม: Admin, Manager, HR, Head Accounting, Marketing ใช้งานในที่ทำงาน บน desktop เป็นหลัก มือถือเป็นรอง งานหลักคือบันทึกข้อมูล ดูรายงาน จัดการออเดอร์และคลังสินค้า ต้องการความรวดเร็วและความชัดเจนในการทำงานประจำวัน

### Brand Personality
**เรียบง่าย · ทันสมัย · มีประสิทธิภาพ**

อารมณ์เป้าหมาย: ผู้ใช้รู้สึก **เร็ว** และ **ง่าย** — ไม่ต้องคิดมาก เข้ามาแล้วทำงานได้ทันที

### Aesthetic Direction
**Reference: Apple** — ความเรียบสะอาด whitespace กว้าง typography ที่อ่านง่าย สี accent เดียว รายละเอียดที่ refined ทุกอย่างมีเหตุผลอยู่บนหน้าจอ ไม่มีองค์ประกอบเกินความจำเป็น

**Anti-reference:** หลีกเลี่ยงสไตล์ที่:
- ดูเป็น AI-generated (gradient เยอะ, สีสดโดยไม่มีเหตุผล, layout ยุ่งเหยิง)
- Palette ไม่เข้ากัน หรือมีสีมากเกินไป
- ข้อมูลหนาแน่นโดยไม่มี breathing room
- Animation หนักหรือ distracting

**Theme:** รองรับทั้ง light และ dark mode เท่ากัน

**Color system:**
- Accent เดียว: `indigo` (indigo-500/600) สำหรับ CTA, active state, focus ring
- Neutral base: gray scale (`gray-50` → `gray-950`) สำหรับทุกอย่างที่เหลือ
- Semantic: `emerald` = success/online, `red` = danger/logout, `amber` = warning
- ห้ามใช้สีที่ไม่อยู่ใน palette นี้โดยไม่มีเหตุผลชัดเจน

**Typography:**
- ขนาดต้องอ่านง่ายในทุก viewport
- Weight hierarchy: `font-bold` (heading) → `font-semibold` (label) → `font-medium` (body) → default (secondary)
- Thai text ต้องให้ line-height เพียงพอ (leading-relaxed หรือมากกว่า)

**Spacing & Shape:**
- Border radius: `rounded-2xl` (cards, buttons, inputs), `rounded-xl` (smaller elements), `rounded-full` (avatars, badges)
- Shadow: `shadow-sm` default, `shadow-md`/`shadow-lg` เฉพาะ hover หรือ elevated
- Padding cards: `p-4` mobile / `p-5`–`p-6` desktop
- Whitespace ใจกว้าง — ให้ content หายใจได้

### Design Principles

1. **Clarity over decoration** — ทุกองค์ประกอบต้องมีเหตุผลที่อยู่บนหน้าจอ ถ้าลบแล้วไม่ขาดอะไร ให้ลบ

2. **One accent, many neutrals** — indigo คือ accent เดียว ใช้เมื่อต้องการนำสายตาหรือบอก action สีอื่นๆ เป็น gray เท่านั้น

3. **Speed through simplicity** — layout ที่ชัดเจนทำให้ผู้ใช้หาสิ่งที่ต้องการได้เร็ว ไม่ต้องอ่านทุกอย่างก่อนกดได้

4. **Consistent rhythm** — ใช้ spacing scale, border-radius, และ shadow level เดิมซ้ำๆ อย่าประดิษฐ์ค่าใหม่โดยไม่จำเป็น

5. **Dark mode as first-class** — ทุกสีต้องผ่านการทดสอบทั้ง light และ dark ไม่มีข้อความที่กลายเป็นสีเดียวกับพื้นหลัง
